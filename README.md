# ISI Op-Ed Research — Strategic Stability and AI

This repository hosts the research draft and publication assets for an op-ed prepared for the **Indus Strategic Institute (ISI)**.

## Suggested repository purpose
- Preserve the canonical manuscript and publication files
- Publish a clean web version through GitHub Pages
- Keep revision history transparent and professional

## Current contents
- `ISI_OpEd_Strategic_Stability_AI_Mohammad Arqam Javed.pages` (Apple Pages source file)
- `index.html` (GitHub Pages landing page)
- `.nojekyll` (prevents Jekyll processing issues)
- `.gitignore` (excludes macOS/system artifacts)

## Recommended publishing workflow
1. Keep the `.pages` file as the master editable source.
2. Export final text from Pages as PDF and (optionally) DOCX.
3. Place exported files in `assets/` (for example: `assets/op-ed.pdf`).
4. Update `index.html` links to point to exported files.
5. Push to GitHub.
6. Enable GitHub Pages from **Settings → Pages** using the `main` branch and `/ (root)`.

## Notes for clean GitHub checks
- Do not commit sensitive data, private emails, access tokens, or confidential institutional material.
- Use only content you own or have rights to publish.
- Keep repository metadata accurate and non-misleading.
- Avoid spam-like automation patterns in commits.
