# ISI Op-Ed: Strategic Stability and Artificial Intelligence

Professional research repository for an op-ed prepared for the Indus Strategic Institute (ISI).

## Repository Summary
- Maintains the manuscript source file
- Publishes a clean public version via GitHub Pages
- Preserves version history for edits and updates

## Files
- `ISI_OpEd_Strategic_Stability_AI_Mohammad Arqam Javed.pages` — master Apple Pages source
- `index.html` — public GitHub Pages landing page
- `assets/` — optional publication assets (PDF, images, references)
- `.nojekyll` — ensures static site compatibility on GitHub Pages
- `.gitignore` — excludes macOS/editor noise

## Publishing Workflow
1. Edit the manuscript in the `.pages` source file.
2. Export final publication as `PDF` (and optional `DOCX`).
3. Add exported files to `assets/` (example: `assets/op-ed.pdf`).
4. Update links in `index.html` if filenames change.
5. Commit and push to `main`.
6. Enable GitHub Pages:
	- Repository **Settings** → **Pages**
	- Source: **Deploy from a branch**
	- Branch: **main** / **(root)**
